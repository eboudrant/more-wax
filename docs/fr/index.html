More'Wax

Vos vinyles. Votre serveur. Vos règles.

Gestionnaire de collection auto-hébergé avec reconnaissance de pochettes par IA, scan de codes-barres et synchronisation Discogs. Tourne sur votre machine, fonctionne sur tout navigateur.

Commencer Voir sur GitHub

Fonctions

Tout ce qu'il faut pour gérer vos galettes.

📷

Scan de codes-barres

Pointez votre caméra vers un code-barres. Quagga.js le détecte en temps réel, puis lance une recherche Discogs pour identifier le pressage exact.

🤖

Reconnaissance de pochette par IA

Prenez en photo n'importe quelle pochette. Claude Vision identifie l'album, l'artiste et le pressage en quelques secondes. Environ 0,007 $ par identification.

💿

Synchronisation Discogs

Importez votre collection Discogs existante avec détection des doublons. Les différents pressages d'un même album sont signalés automatiquement.

💰

Prix du marché

Prix bas, médian et haut en direct depuis la marketplace Discogs. L'actualisation en arrière-plan maintient la valeur de votre collection à jour.

📤

Cartes de partage

Générez des images partageables avec la pochette, les détails de la sortie et un QR code. Partage natif sur mobile, téléchargement sur ordinateur.

🔍

Filtres intelligents

Tapez is: pour filtrer par tracklist manquante, sans prix, doublons ou morceaux aimés. L'autocomplétion fantôme vous guide.

🔒

Google OAuth

Protégez votre collection lorsqu'elle est exposée en ligne. Accès mono-utilisateur verrouillé dès la première connexion. Authentification contournée automatiquement sur le réseau local.

🏠

100 % privé

Auto-hébergé. Pas de cloud. Pas de compte. Vos données restent sur votre machine. Serveur Python stdlib sans aucune dépendance pip.

Installation

Opérationnel en 30 secondes.

Un assistant de configuration vous guide pour connecter votre compte Discogs au premier lancement.

Docker (recommandé)

docker run -d \
  --name more-wax \
  -p 8765:8765 -p 8766:8766 \
  -v morewax-data:/app/data \
  eboudrant/more-wax:latest

Puis ouvrez https://localhost:8766

Docker Compose

docker compose up -d

Puis ouvrez https://localhost:8766

Depuis les sources

git clone https://github.com/eboudrant/more-wax.git
cd more-wax
python3 server.py

Puis ouvrez https://localhost:8766

Publier en ligne

Accessible de partout.

Placez More'Wax derrière un reverse proxy pour y accéder hors de votre réseau local. Google OAuth protège automatiquement votre collection.

Cloudflare Tunnel

cloudflared tunnel --url http://localhost:8765

Aucune redirection de port nécessaire. Offre gratuite disponible.

Caddy (HTTPS automatique)

wax.yourdomain.com {
    reverse_proxy localhost:8765
}

Certificats Let's Encrypt automatiques.

Google OAuth

Settings → Authentication
→ Paste Client ID & Secret
→ Enable Authentication

La première connexion devient propriétaire. Authentification contournée sur le réseau local.
