More'Wax

Deine Platten. Dein Server. Deine Regeln.

Selbstgehosteter Sammlungs-Manager mit KI-Covererkennung, Barcode-Scanner und Discogs-Sync. Läuft auf deinem Rechner, funktioniert in jedem Browser.

Loslegen Auf GitHub ansehen

Funktionen

Alles, was du brauchst, um deine Platten zu verwalten.

📷

Barcode-Scan

Halte deine Kamera auf einen Barcode. Quagga.js erkennt ihn in Echtzeit und sucht die genaue Pressung auf Discogs.

🤖

KI-Covererkennung

Fotografiere ein beliebiges Cover. Claude Vision erkennt Album, Interpret und Pressung in Sekunden. ~0,007 $ pro Erkennung.

💿

Discogs-Sync

Importiere deine bestehende Discogs-Sammlung mit Duplikaterkennung. Verschiedene Pressungen desselben Albums werden automatisch markiert.

💰

Marktplatz-Preise

Aktuelle Niedrig-, Median- und Höchstpreise vom Discogs-Marktplatz. Hintergrund-Aktualisierung hält den Wert deiner Sammlung auf dem neuesten Stand.

📤

Teilen-Karten

Erstelle teilbare Bilder mit Cover, Release-Details und QR-Code. Natives Teilen auf dem Handy, Download am Desktop.

🔍

Smarte Filter

Tippe is: um nach fehlender Tracklist, fehlendem Preis, Duplikaten oder markierten Tracks zu filtern. Ghost-Text-Autocomplete hilft dir dabei.

🔒

Google OAuth

Schütze deine Sammlung, wenn sie online erreichbar ist. Einzelnutzer-Sperre beim ersten Login. Authentifizierung wird im LAN automatisch umgangen.

🏠

100 % privat

Selbstgehostet. Keine Cloud. Kein Konto. Deine Daten bleiben auf deinem Rechner. Python-Stdlib-Server ohne pip-Abhängigkeiten.

Loslegen

In 30 Sekunden startklar.

Ein Einrichtungsassistent führt dich beim ersten Start durch die Verbindung mit deinem Discogs-Konto.

Docker (empfohlen)

docker run -d \
  --name more-wax \
  -p 8765:8765 -p 8766:8766 \
  -v morewax-data:/app/data \
  eboudrant/more-wax:latest

Dann öffne https://localhost:8766

Docker Compose

docker compose up -d

Dann öffne https://localhost:8766

Aus dem Quellcode starten

git clone https://github.com/eboudrant/more-wax.git
cd more-wax
python3 server.py

Dann öffne https://localhost:8766

Online stellen

Von überall zugreifen.

Stelle More'Wax hinter einen Reverse Proxy, um von außerhalb deines LANs darauf zuzugreifen. Google OAuth schützt deine Sammlung automatisch.

Cloudflare Tunnel

cloudflared tunnel --url http://localhost:8765

Keine Portweiterleitung nötig. Kostenlose Stufe verfügbar.

Caddy (automatisches HTTPS)

wax.yourdomain.com {
    reverse_proxy localhost:8765
}

Automatische Let's-Encrypt-Zertifikate.

Google OAuth

Settings → Authentication
→ Paste Client ID & Secret
→ Enable Authentication

Erste Anmeldung wird zum Besitzer. Authentifizierung wird im LAN umgangen.
